Blood Type Compatibility

Understanding blood type compatibility is crucial for safe transfusions. Here's what you need to know:

A+

Type A+

Donates to: A+, AB+
Receives from: A+, A-, O+, O-
Population: ~34%
A-

Type A-

Donates to: A+, A-, AB+, AB-
Receives from: A-, O-
Population: ~6%
B+

Type B+

Donates to: B+, AB+
Receives from: B+, B-, O+, O-
Population: ~9%
B-

Type B-

Donates to: B+, B-, AB+, AB-
Receives from: B-, O-
Population: ~2%
AB+

Type AB+

Donates to: AB+ only
Receives from: All types
Population: ~3%
AB-

Type AB-

Donates to: AB+, AB-
Receives from: A-, B-, AB-, O-
Population: ~1%
O+

Type O+

Donates to: O+, A+, B+, AB+
Receives from: O+, O-
Population: ~38%
O-

Type O-

Donates to: All types
Receives from: O- only
Population: ~7%

Key Facts:

  • O- is the universal donor (can donate to anyone)
  • AB+ is the universal recipient (can receive from anyone)
  • Blood type is determined by genetics
  • Rh factor (+ or -) is also important for compatibility
